# TP 6 SIR - Pokedex :

###### Ce TP comporte 3 fichiers importants que nous avons réalisés : 

** - pokedex.js ** : Composé de code JS utilisant Angular, il contient 2 Controller et 1 Factory
Controllers : Récuperation et Affichage des informations
Factory : Requête vers l'API pokeAPI

** - index.html ** : Page permettant l'éxécution du projet (utilise le code de pokedex.html via une directive)

** - pokedex.html ** : Page qui fournit a index.html les informations, en plusieurs parties, Recherche du Pokemon, Informations du Pokemon, Statistiques, Abilites, Moves.
